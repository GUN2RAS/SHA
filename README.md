# SHA

Sodium Hologram API. A small Fabric library that lets other mods render fake blocks and entities client-side without actually placing them in the world.

Used as a base for visual effects like portals, previews, and structure holograms. No physics by default, just rendering through Sodium.

## Build

```bash
./gradlew build
```

Jar goes to `build/libs/`.

## Stack

Minecraft 26.1.2, Fabric, Java 25, Mojmap, Sodium.
